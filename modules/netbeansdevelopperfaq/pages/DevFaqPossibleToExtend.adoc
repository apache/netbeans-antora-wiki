= Is it possible to extend an existing project type?
:page-layout: wikimenu
:page-tags: wiki, devfaq
:jbake-status: published
:keywords: Apache NetBeans wiki DevFaqPossibleToExtend
:description: Apache NetBeans wiki DevFaqPossibleToExtend
:toc: left
:toc-title:
:page-syntax: true
:page-wikidevsection: _project_types
:page-position: 1
:page-aliases: ROOT:wiki/DevFaqPossibleToExtend.adoc

Yes.

You can add new actions, tabs, nodes, and lookups to an existing project type. See the xref:tutorial::tutorials/nbm-projectextension.adoc[NetBeans Project Type Extension Module Tutorial] for details.
