Use the link:{apidoclink}org-netbeans-modules-options-api/overview-summary.html[Options Dialog and SPI] as explained in the xref:tutorial::tutorials/nbm-options.adoc[NetBeans Options Window Module Tutorial].

There are wizards for creating primary and secondary options panels. Use `New File.../Module Development/Options Panel` and follow the wizard.
